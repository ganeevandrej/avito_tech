# Avito Модерация

Приложение для модерации объявлений Avito.

## Запуск с Docker

```bash
docker-compose up --build
```

- Клиент: http://localhost:5173
- API: http://localhost:3000

## Локальный запуск

### Сервер

```bash
cd server
npm install
npm run dev
```

### Клиент

```bash
cd client
npm install
npm run dev
```

- Клиент: http://localhost:5173
- API: http://localhost:3000
